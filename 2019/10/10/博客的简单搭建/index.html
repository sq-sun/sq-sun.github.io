博客的简单搭建

发表于 2019-10-10 阅读次数： Disqus：

下载NodeJS

查看nodejs 版本：Node -v

安装cnpm

npm install -g cmpm –registry=https://registry.mpm.taobao.org
或者 npm install hexo-cli -g 完成后直接初始化

安装hexo(基于node)

cnpm install -g hexo-cli

创建一个blog文件夹

初始化博客[用管理员权限打开cmd] blog目录下

hexo init (安装完毕后会提示Strart blogging with Hexo! 文件夹内生成若干文件)

启动博客

hexo s

部署到远端

首先登陆GitHub 新建仓库。仓库命名规范：用户名.github.io
在blog文件夹中安装git部署插件
npm install –save hexo -deployer-git
编辑文件 _config.yml
在文件最底部 type: git
repo: 仓库地址（GitHub复制）
branch: master
接下来部署到远端
hexo clean
hexo g
hexo d (会提示输入Github账号)

部署到远端可能出现的问题

如果中途报错，是git未设置global身份，在git bash here中设置身份
设置身份： git config –global user.name ‘xxx’
git config –global user.email “xxx@qq.com“
查询身份： git config user.name
git config user.email

启用主题

克隆主题：git clone http://github.com/theme-next/hexo-theme-next themes/next
配置_config.yml文件：

完成，记得要推到远端。
